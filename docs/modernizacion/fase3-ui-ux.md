# Fase 3 · Sistema UX/UI propuesto

## Principios
- Coherencia institucional global.
- Identidad visual específica por observatorio.
- Datos primero: lectura rápida + exploración profunda.
- Accesibilidad AA y lenguaje ciudadano.

## Design system base
- **Tokens**: color, tipografía, spacing, radios, sombras.
- **Componentes**: header, breadcrumb, tabs, cards KPI, tablas, filtros, alertas, badges de estado.
- **Plantillas**: home red, home observatorio, ficha indicador, listado noticias, repositorio documental, panel admin.

## Identidad por observatorio (paletas)
- Económico: azul petróleo + acento ámbar.
- Social: morado + magenta suave.
- Ambiente: verde bosque + cian.
- CTI: azul eléctrico + violeta.
- Género: púrpura + coral.

## Wireframe descriptivo (texto)
1. **Home Red**: hero + buscador global + 5 tarjetas de observatorio + cifras de red + actualidad + CTA admin.
2. **Home observatorio**: hero temático + KPIs + indicadores destacados + noticias + documentos/datasets + mapa futuro.
3. **Login**: formulario simple, recuperación y mensajes claros.
4. **Admin dashboard**: pendientes de revisión, actividad reciente, publicaciones por estado y accesos rápidos.
